# TaskFlow – SaaS Landing Page

TaskFlow is a modern, responsive SaaS landing page built with **Tailwind CSS**.  
The project focuses on clean layout design, reusable UI patterns, and smooth user experience across devices.

This is my **4th Tailwind CSS project**, completed in ~2.5 days, with an emphasis on layout, spacing, responsiveness, and component structure.

## 🚀 Live Features

- Responsive navigation (desktop & mobile)
- Hero section with clear value proposition
- Features section with visual illustrations
- Testimonials section
- Pricing plans (Free, Pro, Team)
- Call-to-action section
- Fully responsive footer
- Smooth scrolling navigation
- Mobile menu toggle with JavaScript
- Subtle gradients and hover animations

---

## 🛠 Tech Stack

- **HTML5**
- **Tailwind CSS (CDN)**
- **Vanilla JavaScript**
- **Google Fonts (Inter)**
- **Boxicons**

---

## 🎨 Design Decisions

- **Tailwind CSS** used for rapid UI development and consistent spacing
- Flexbox & Grid combined where appropriate:
  - Grid for page-level layouts
  - Flexbox for alignment and card content
- Consistent card heights using `flex` + `mt-auto`
- Gradient buttons for primary actions
- Mobile-first responsive approach

---

## 📱 Responsive Behavior

- Desktop: multi-column layouts with spacing balance
- Tablet: adaptive grids and centered content
- Mobile:
  - Collapsible navigation menu
  - Stacked sections
  - Touch-friendly buttons
